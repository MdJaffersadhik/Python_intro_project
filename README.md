# Personal Introduction Program (Week 1 Python Project)

## Project Overview
This project collects user information such as name, age, hobby, city, and favorite food.  
After collecting the inputs, it displays a friendly personalized message.  
This is my Week 1 Python project for learning programming basics.

## Project Objectives
- Understand basic Python syntax  
- Learn how to use `input()` to get user data  
- Store answers in variables  
- Display output using `print()` and f-strings  
- Practice writing a simple Python script  
- Learn how to upload files to GitHub  


## Setup Instructions

1. Install Python  
Download from https://www.python.org and install.

2. Verify Installation

3. Run the Program

4. File Structure

## Code Structure
- **personal_intro.py** → Main Python program  
- **requirements.txt** → Python version details  
- **screenshot.png** → Screenshot of program running  
- **README.md** → Documentation of the project  


# Technical Details
## Input Validation
The program checks if the user enters empty input.  
If the input is blank, it asks again.

# Concepts Used
- Variables  
- Input/output  
- f-strings  
- Basic string formatting  

# Program Flow
1. Ask for name  
2. Ask for age  
3. Ask for hobby  
4. Ask for city  
5. Ask for favorite food  
6. Print summary message
